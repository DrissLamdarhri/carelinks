## 📊 Data Flow

### Approval Flow
```
Admin Panel
    ↓ clicks "Approuver"
Details Modal
    ↓ confirmation
Supabase
    ├── UPDATE professionals (status='approved')
    ├── INSERT notifications (type='approval')
    └── CALL send-approval-email
         ├── Fetch Resend API
         ├── Send HTML email
         └── Return success
    ↓
UI Update
    ├── Close modal
    ├── Refresh list
    └── Show toast: "Professionnel approuvé"
```

### Rejection Flow
```
Admin Panel
    ↓ clicks "Rejeter"
Rejection Modal
    ↓ enters reason
Confirmation
    ↓ clicks "Confirmer le Rejet"
Supabase
    ├── UPDATE professionals (status='rejected', reason=...)
    ├── INSERT notifications (type='rejection', message=...)
    └── CALL send-rejection-email
         ├── Fetch Resend API
         ├── Send HTML email with reason
         └── Return success
    ↓
UI Update
    ├── Close modals
    ├── Refresh list
    └── Show toast: "Professionnel rejeté"
```
